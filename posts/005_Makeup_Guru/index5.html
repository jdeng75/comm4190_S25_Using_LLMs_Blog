Can Generative AI find me the best makeup mentor?

Recently, I’ve been getting more interest in makeup because my friends all around me wear it. Whether to hide a pimple, dark underseyes, or just put more glitter on their faces, I want to also understand how to use certain products to acheive a specific look. But I need help. Youtube videos are too long, blog posts don’t give enough visuals, and I just don’t know anyone on the internet who looks similar to me. I don’t know what my foundation type is, I don’t know what blush color looks the best, and I don’t want to be spending well over $100 just to acheive a natural makeup look.

I’m interested to see whether I can create 5 different SVGs of makeup artists and have them give tips for certain pictures of models. Maybe then, I can generate tips for how I should go about wearing makeup.

Show Code

import svgwrite

def create_makeup_artist_svg(filename, face_color, hair_color, lips_color, eyeshadow_color): dwg = svgwrite.Drawing(filename, profile=‘tiny’, size=(“200px”, “200px”))

# Face
dwg.add(dwg.circle(center=(100, 100), r=50, fill=face_color))

# Hair
dwg.add(dwg.rect(insert=(50, 40), size=(100, 80), fill=hair_color, rx=20, ry=20))

# Eyes
dwg.add(dwg.circle(center=(80, 90), r=5, fill="black"))
dwg.add(dwg.circle(center=(120, 90), r=5, fill="black"))

# Eyeshadow
dwg.add(dwg.ellipse(center=(80, 85), r=(10, 5), fill=eyeshadow_color, opacity=0.6))
dwg.add(dwg.ellipse(center=(120, 85), r=(10, 5), fill=eyeshadow_color, opacity=0.6))

# Lips
dwg.add(dwg.ellipse(center=(100, 120), r=(15, 8), fill=lips_color))

# Save SVG
dwg.save()

import cv2 import numpy as np from PIL import ImageEnhance # Load image using OpenCV for processing image_cv = cv2.imread(image_path) image_rgb = cv2.cvtColor(image_cv, cv2.COLOR_BGR2RGB) # Function to apply different makeup styles def apply_makeup_style(image, style): img = image.copy() if style == “Sophia Glam”: # Bridal Makeup img = cv2.GaussianBlur(img, (5, 5), 1) # Smooth skin img = cv2.addWeighted(img, 1.1, np.zeros(img.shape, img.dtype), 0, 10) # Slight glow img[:, :, 2] = np.clip(img[:, :, 2] + 20, 0, 255) # Rosy lips elif style == “Luna Luxe”: # Editorial Makeup img[:, :, 0] = np.clip(img[:, :, 0] + 30, 0, 255) # Enhance blue tones for bold eyeshadow img[:, :, 1] = np.clip(img[:, :, 1] + 10, 0, 255) # Add subtle green undertone img = cv2.addWeighted(img, 1.2, np.zeros(img.shape, img.dtype), 0, 15) # Glossy highlight elif style == “Jade Beauty”: # Theatrical SFX img[:, :, 0] = np.clip(img[:, :, 0] + 50, 0, 255) # Intense eye contouring img = cv2.applyColorMap(img, cv2.COLORMAP_JET) # Fantasy-like effect elif style == “Mia Elegance”: # Red Carpet Glam img[:, :, 2] = np.clip(img[:, :, 2] + 40, 0, 255) # Bold lips img = cv2.addWeighted(img, 1.15, np.zeros(img.shape, img.dtype), 0, 10) # Sculpted contour lif style == “Ella Artistry”: # Everyday Natural img = cv2.GaussianBlur(img, (3, 3), 1) # Soft focus img[:, :, 1] = np.clip(img[:, :, 1] + 15, 0, 255) # Subtle glow return Image.fromarray(img)

#Generate makeup variations makeup_styles = [“Sophia Glam”, “Luna Luxe”, “Jade Beauty”, “Mia Elegance”, “Ella Artistry”] makeup_images = {style: apply_makeup_style(image_rgb, style) for style in makeup_styles}

#Save and display the results output_paths = {} for style, img in makeup_images.items(): output_path = f”/mnt/data/{style.replace(’ ‘,’_’).lower()}_makeup.png” img.save(output_path) output_paths[style] = output_path #Display the generated images to the user import ace_tools as tools tools.display_dataframe_to_user(“Makeup Style Transformations”, output_paths)

Well, how disappointing.



Nothing was done to modify the model’s face, there were only filters being added into the photo. We saw this in a couple of class examples though. Generative AI has a hard time with images - whether it’s trying to put words into image creation or interpreting how to create the image. I remember in class, we couldn’t get ChatGPT to create a workout diagram of someone on their side.

Perhaps I should just stick to human makeup mentors.
